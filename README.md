# arcdps universal loader

Tired of your arcdps addons not working because GW2 updated and arcdps is broken?
This loader will load your arcdps addons when arcdps is unavailable (as long as the addons do not require combat data).

## Features

Supports being loaded by:

1. [Nexus](https://raidcore.gg/gw2/nexus)
1. [addon-loader](https://github.com/gw2-addon-loader/GW2-Addon-Manager)
1. [GW2Load](https://github.com/GW2Load/GW2Load)
1. As a `d3d11.dll` proxy
1. As a `dxgi.dll` proxy

## Installation

1. Download the latest release from the [releases page](https://github.com/arcdps-universal-loader/arcdps-universal-loader/releases)
1. Install it according to the loader you are using
